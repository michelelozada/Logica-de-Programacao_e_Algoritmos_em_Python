### 1. Adição
```py

a = 4
b = 2

print(a + b)  # Retorna: 6
```

&nbsp; 

### 2. Subtração
```py

print(a - b)  # 2
```

&nbsp; 

### 3. Multiplicação
```py

print(a * b)  # 8
```

&nbsp; 

### 4. Divisão (retorna valor decimal real)
```py

print(a / b)  # 2.0
```

&nbsp; 

### 5. Divisão (retorna valor inteiro)
```py

print(a // b)  # 2
```

&nbsp; 

### 6. Módulo (retorna o resto da divisão)
```py

print(a % b)  # 0
```

&nbsp; 

### 7. Exponencial
```py

print(a ** b) # 16
```
