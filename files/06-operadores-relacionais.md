&nbsp;  

### 1 - Igualdade (==)
```py

print(4 == 4.0)  # Retorna: True
```

&nbsp;  

### 2 - Diferente de (!=)
```py

print(4 != 4.0)  # False
```

&nbsp;  

### 3 - Maior que (>)
```py

print(5 > 3) # True
```

&nbsp;  

### 4 - Menor que (<)
```py

print(2.5 < 10) # True
```

&nbsp;  

### 5 - Maior ou igual a (>=)
```py

print(5 >= 13) # False
print(5.0 >= 5) # True
```

&nbsp;  

### 6 - Menor ou igual a (<=)
```py

print(10 <= 5) # False
print(10 <= 10.0) # True
```

&nbsp;  

### 7 - Comparando strings
```py

print('bom' < 'dia') # True
print('bom' == 'dia') # False
print('bom' > 'dia') # False
print('dia' == 'dia') # True
```
